#  游戏玩法：<br>
长按鼠标中键移动屏幕，鼠标左键编辑，鼠标右键删除，鼠标左键长按拖动<br>
菜单栏中鼠标左键选中，鼠标右键切换菜单 鼠标中键切换远景物体 <br>
回车键创建关卡<br> 
Esc键返回编辑器 ← →控制玩家移动 空格键跳跃<br>
